Welcome to Affinity!

For an overview of the philosophy behind Affinity, visit the FAQ. There's also a list of features, and an assessment of Affinity's comparative strengths.

The page on terminology can be used as introductory material, to assimilate the terms and concepts in Affinity. In particular, it quickly describes the data model, the interfaces and the software components in the Affinity package. This page also cements the rest of the documentation, which points to it as a unified reference. Finally, it contains an alphabetically ordered list of terms, for quick lookup.
